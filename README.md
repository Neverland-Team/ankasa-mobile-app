### # HOW TO INSTALL

Download the projects:

```
~$ git clone https://github.com/Neverland-Team/ankasa-mobile-app
```

install dependencies using npm or yarn   
```
yarn OR npm i
```

running App
```
yarn android or react-native run-android
```
